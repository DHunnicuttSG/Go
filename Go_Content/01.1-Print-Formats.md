Here is a **clean, reference-style table of the most important Go `fmt` formatting verbs** used 
with `Printf`, `Sprintf`, `Fprintf`, etc.

Think of them as tools for telling Go *how* to display a value.

---

## ðŸ”¹ Universal / Default Verbs

| Verb  | Meaning                  | Example                             |
| ----- | ------------------------ | ----------------------------------- |
| `%v`  | Default format           | `fmt.Printf("%v", x)`               |
| `%+v` | Struct with field names  | `{Name:John Age:30}`                |
| `%#v` | Go-syntax representation | `main.Person{Name:"John", Age:30}`  |
| `%T`  | Type of the value        | `int`, `[]string`, `map[string]int` |
| `%%`  | Prints a literal `%`     | `%`                                 |

---

## ðŸ”¹ Integer Verbs

| Verb | Meaning            | Example  |
| ---- | ------------------ | -------- |
| `%d` | Base 10 integer    | `42`     |
| `%b` | Binary             | `101010` |
| `%o` | Octal              | `52`     |
| `%x` | Hex (lowercase)    | `2a`     |
| `%X` | Hex (uppercase)    | `2A`     |
| `%U` | Unicode format     | `U+002A` |
| `%c` | Character from int | `*`      |
| `%q` | Quoted character   | `'*'`    |

Example:

```go
fmt.Printf("%d %b %x\n", 42, 42, 42)
```

---

## ðŸ”¹ Floating Point Verbs

| Verb | Meaning             | Example    |
| ---- | ------------------- | ---------- |
| `%f` | Decimal notation    | `3.140000` |
| `%F` | Same as `%f`        | `3.14`     |
| `%e` | Scientific (e)      | `3.14e+00` |
| `%E` | Scientific (E)      | `3.14E+00` |
| `%g` | Compact form        | `3.14`     |
| `%G` | Compact (uppercase) | `3.14`     |

Example:

```go
fmt.Printf("%f %e %g\n", 3.14, 3.14, 3.14)
```

---

## ðŸ”¹ String & Slice of Bytes Verbs

| Verb  | Meaning                 | Example          |
| ----- | ----------------------- | ---------------- |
| `%s`  | String                  | `hello`          |
| `%q`  | Quoted string           | `"hello"`        |
| `%x`  | Hex of bytes            | `68656c6c6f`     |
| `% X` | Hex (spaced, uppercase) | `68 65 6C 6C 6F` |

Example:

```go
fmt.Printf("%s %q %x\n", "hi", "hi", []byte("hi"))
```

---

## ðŸ”¹ Pointer Verbs

| Verb | Meaning                  | Example        |
| ---- | ------------------------ | -------------- |
| `%p` | Memory address (pointer) | `0xc00001e0c0` |

Example:

```go
fmt.Printf("%p", &x)
```

**Important:** `%p` only works with **pointers** (`&x`, not `x`).

---

## ðŸ”¹ Boolean

| Verb | Meaning           | Example |
| ---- | ----------------- | ------- |
| `%t` | `true` or `false` | `true`  |

```go
fmt.Printf("%t", true)
```

## ðŸ”¹ Structs / Maps / Slices

Youâ€™ll mainly use:

| Best practice | Why                  |
| ------------- | -------------------- |
| `%v`          | Human readable       |
| `%+v`         | Includes field names |
| `%#v`         | Debug / Go-syntax    |

Example:

```go
type Person struct {
   Name string
   Age int
}
p := Person{"Sam", 40}

fmt.Printf("%v\n", p)   // {Sam 40}
fmt.Printf("%+v\n", p)  // {Name:Sam Age:40}
fmt.Printf("%#v\n", p)  // main.Person{Name:"Sam", Age:40}
```

---

## ðŸ”¹ Width & Precision

You can control output alignment and size:

| Syntax  | Purpose         | Example   |
| ------- | --------------- | --------- |
| `%6d`   | Min width 6     | `    42`  |
| `%06d`  | Pad with zeros  | `000042`  |
| `%.2f`  | 2 decimals      | `3.14`    |
| `%8.2f` | Width+precision | `   3.14` |
| `%-6s`  | Left-align      | `hi    `  |

Example:

```go
fmt.Printf("|%8.2f|\n", 3.14)
fmt.Printf("|%-10s|\n", "Go")
```

## ðŸ”¹ Real-world cheat block (copy/paste)

```go
fmt.Printf("%v\n", x)   // default
fmt.Printf("%+v\n", x)  // struct with fields
fmt.Printf("%#v\n", x)  // Go syntax
fmt.Printf("%T\n", x)   // type
fmt.Printf("%p\n", &x)  // pointer
fmt.Printf("%d\n", 42)
fmt.Printf("%f\n", 3.14)
fmt.Printf("%s\n", "Go")
fmt.Printf("%t\n", true)
fmt.Printf("%%\n")       // print %
```
